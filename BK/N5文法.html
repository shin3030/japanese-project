第一課


A は B です (A是B)

は：助詞，讀音為 wa

です：表肯定

A は B ではありません (A不是B)

ではありません：表否定

ではありません ＝ じゃありません＝ではないです

では＝じゃ

ありません＝ない

A は B ですか？ (A是B嗎?)

か：疑問助詞

  • Yes --> はい、A は B です
  • No --> いいえ、A は B ではありません

    • *這裡的 A,B 皆放入名詞

    敬體與常體之間肯定與否定的差異
    敬體 常體
    肯定 はい、A は B です。 はい、A は B だ。
    否定 いいえ、A は B ではありません。 いいえ、A は B ではない。
    いいえ、A は B ではないです。
    否定(較口語) いいえ、A は B じゃありません。 いいえ、A は B じゃない。

    指示詞

    在此假設自己為說話者，另一個人為聽話者，在有點距離的情況下。

    用法 意思 距離 解釋
    これ　この　ここ 這 近距離 離自己較近的事物或場所
    それ　その　そこ 那 中距離 離對方較近的事物或場所
    あれ　あの　あそこ 那 遠距離 離雙方都很遠的事物或場所
    どれ　どの　どこ 哪一個 距離不確定 指三件以上人事物之其中一個

  • これ／それ／あれ　＋　は　＋　N　＋です
  • これ／それ／あれ　＋　は　＋ S　＋　の　＋　N　＋　です
  • この／その／あの　＋　N１(物）＋　は　＋　 S　＋　の　＋（N1）＋　です
  • N　＋　は　＋　どれ　＋　ですか
  • N　＋　は　＋　どの　＋　N　＋　ですか

    • 例：図書館はどの建物ですか。

    ★

    どれ問　→　これ／それ／あれ答

    どの問　→　この／その／あの答

    依照距離提問語回答

  • 物品物品兩人皆靠近：問：これ／この　→　答：これ／この
  • 物品離A有段距離，離B比較近：
    A問：それ／その　B答：これ／この
  • 物品離兩人都遠：問：あれ／あの　→　答：あれ／あの

    • 依照距離提問語回答

  • 自己&對方所在的場所：問：ここ　→　答：ここ
  • 離A有點距離的場所，離B較近：
    A問：そこ　B答：ここ
  • 離自己&對方都遠的場所：問：あそこ　→　答：あそこ

    • 助詞の

    N1 の N2 (某人的東西)

    N1為人，N2 為事物

    助詞も

    N1 も N2 です(某人也是...)

    表示並例，中文"也"的意思。

    第二課


    い形容詞

    い形容詞 + 名詞

    形容詞以い結尾

    な形容詞

    形容動詞 + な + 名詞

    例：綺麗な部屋、嫌いな野菜

    *綺麗（きれい）、嫌い（きらい）這兩字雖然是い結尾，但卻是な形容詞

    S （主語） は ＋い形容詞＋N（名詞）＋です
    S（主詞）は＋な形容詞＋な＋N（名詞）＋です
    S（主語） は A ですか、 Bですか

    *A, B皆放入形容詞


    有無句

    A に B があります / います（A有B）

    A放場所、位置

    B放存在的東西或人...

    B是有生命（會動）的話使用います

    B是無生命（無法自行移動）的話使用あります


    に：助詞，表場所、地點

    が：助詞，表有無

    助詞　と

    と：表並列關係

    A に B があります / いますか？（A有B嗎？）

    A放場所、位置

    B放入疑問詞

  • 需注意B放入的為無生命或是有生命的人或是動物...
    要跟後方的あります、います做搭配

    • 例：だれがいますか、なにがありますか、なにがいますか

    あります / います的肯定否定

    肯定 否定
    います いません
    あります ありません

  • Yes--->はい、B（名詞）があります／います。
  • No--->いいえ、B（名詞）はありません／いません。

    • 存在句

    A には B もありません / いません (A不在B)

    A放場所、位置

    B放入疑問詞

  • 需注意B放入的為無生命或是有生命的人或是動物...
    要跟後方的あります、います做搭配
    • A は B にいます / あります（A 在 B）

    A放東西、人...

    B放場所、位置

  • Question:A （物／人）は＋ どこ＋に＋ありますか／いますか
  • Answer: A（物／人）は＋B＋に＋あります／います

    • 有無疑問句 VS. 存在疑問句

    場所＋に＋何／誰＋が＋ありますか／いますか

    物／人＋は＋どこ+に＋ありますか／いますか

    ★

    疑問詞+が

    は＋疑問句


    ～や～（など）

    助詞，表列舉

    N1 や N2 など

    N1 や N2 や N3 など

    四、動詞

    (あ)V1 未然形：否定形(ない形)

    (い)V2 連用形(ます形)／過去形(た形)／中止形(て形)

    (う)V3 辭書形：動詞原形

    (う)V4 連體形：當形容詞用，後接名詞。う段 + 名詞單字

    (え)V5 可能形／假定形(ば形)

    (え)V6 命令形

    (お)V7 意志形：在意志的表現上會變成無視聽者的自言自語。還有，對熟識的聽者使用的話，
    則可表示勸誘，或者聲明自己的行為。意志形加上「とする」「とします」的話，
    則表示出某行為嘗試過但是未達成，或者表示出事情在實現之前的狀態。

    常體 敬體
    現在
    未來    		 肯定 V3 V2 + ます
    否定 V1 + ない V2 + ません
    過去 肯定 V2 + た V2 + ました
    否定 V1 + なかった V2 + ませんでした

    *動詞分類

    種類 區別方法（看動詞的語尾和語尾前面一個字的發音）
    一類 五段動詞 ①終止形不是 る 音為結尾
    ②終止形不是 る 音為結尾、但 る 前一個字的發音在
    ア 段音、ウ 段音、オ 段音
    ③ 有些字外形看起來像是上下一段動詞，但卻是五段動詞
    二類 上一段動詞 終止形必為 る 音為結尾、但 る 前一個字的發音在 イ 段音
    下一段動詞 終止形必為 る 音為結尾、但 る 前一個字的發音在 エ 段音
    三類 カ行變格動詞 簡稱"カ變動詞"，語尾變化並不規則。
    在日語中只有　くる　一個動詞
    サ行變格動詞 簡稱"サ變動詞"，語尾變化並不規則。
    在日語中只有　する　一個動詞

    音便-->為了發音方便

    音便只會發生在五段動詞

    *當五段動詞V2 下面接助動詞的 「たら」「た」、
    或是接助詞的 「たり」「て」「ては」「ても」...時

    (た形)(て形)

    カ行 バ行 ナ行 マ行 タ行 ワ行 ラ行
    動詞結尾 く ぐ ぶ ぬ む つ う る
    い音便 書く 泳ぐ
    ん音便 遊ぶ 死ぬ 飲む
    つ音便 立つ 笑う ある

    【い音便】　書く　→　書「き」た　→　書「い」た 　　
    泳ぐ　→　泳「ぎ」た　→　泳「い」だ

    【鼻音便】　遊ぶ　→　遊「び」た　→　遊「ん」だ 　　　
    死ぬ　→　死「に」た　→　死「ん」だ 　　　
    飲む　→　飲「み」た　→　飲「ん」だ

    【促音便】　立つ　→　立「ち」た　→　立「っ」た 　　　
    笑う　→　笑「い」た　→　笑「っ」た 　　　
    ある　→　あ「り」た　→　あ「っ」た

    「行く」是五段動詞裡面唯一音便的例外

    行く　→　行きた　→　行いた（X）　行く　→　行った（〇）

    第六課


    A：（給与者）　B：（接受者）　C：（旁観者）

    あげる　(あげます）(給)

    (人) は (人) に N(名詞) をあげる

    もらう（もらいます）(得到)

    (人) は (人) に N(名詞) をもらう

    くれる（くれます）(某人給我 或我的家人)

    (人) は (わたし或是 わたしの家人) に N(名詞) をくれる


    可否 説話者 動作者 対方 対象 動作者的動作 中訳
    OK A説： 私は Bに 本を あげます。 我送給B一本書
    X B説： Aは 私に 本を あげます A送給我一本書
    OK C説： Aは Bに 本を あげます。 A送給B一本書
    X A説： Bは 私に 本を もらいます。 B從我這裡得到一本書
    OK B説： 私は Aに（から） 本を もらいます。 我從A那裡得到一本書
    OK C説： Bは Aに（から） 本を もらいます。 B從A那裡得到一本書
    X A説： 私は Bに 本を くれます。 我送給B一本書
    OK B説： Aは 私に 本を くれます。 A送我一本書
    *註1 C説： Aは Bに 本を くれます。 A送給B一本書

    「くれる」原則上只使用在別人給自己的情況，但如果
    AはBに本をくれます。接受者B是我方的人，這時就可以使用くれる。

    （１）学校の友達が私の家族にお土産をくれました。

    註１：原則上第三者描述給予句子的時候只能使用「あげる」和「くれる」
    但若是出現了像下面（２）這樣的句子時，就可判斷出マリアさん是和自己比較親近的人。

    （２）ジョンさんがマリアさんに花をくれました。

    先輩、同僚、後輩　の使い方

    「あげる」系列

    先輩：差（さ）し上げる　　同僚：あげる　　後輩や動物や植物：やる
